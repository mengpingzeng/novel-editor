## 一、四阶段产出检查表

### 阶段一产出检查（结构扫描）
- [ ] 章节统计表（总章数、字数范围、中位数、字数曲线趋势）
- [ ] 卷/部划分表（每卷起止章+核心冲突）
- [ ] 全书转折点已标注

### 阶段二产出检查（深度抽样·v6 升级为双轴采样）
- [ ] ★ 阶段二·双轴采样：是否每卷取了起/承/转/合各1章（承段取最接近均值章）？
- [ ] ★ 阶段二·双轴采样：是否从全书取了温情峰Top2+冲突谷Top2+共现谷Top2+对话峰Top2？
- [ ] ★ 阶段二·总采样：是否达到18-22章？
- [ ] 每个抽样章是否包含：段落功能标注、对话占比精确值、爽点标注、五段式映射、钩子分析？
- [ ] 是否完成了抽样章横向对比？

### 阶段三产出检查（全篇模式提取）
- [ ] 爽点分类是否每类都含完整触发公式（不能只列名称）？
- [ ] 人物模板是否明确标注了 `{底层逻辑}` 和 `[表层变量]`？
- [ ] 节奏参数是否有抽样章数据做量化支撑（不能纯靠估计）？
- [ ] 文风分析是否含精确的百分比数据（不是"较多""较少"等模糊描述）？
- [ ] ★ 社会语言层次模型：是否至少提取 3 层？每层 4 个维度是否全部填充？
- [ ] ★ 角色语言指纹库：是否至少提取 3 个角色？每个角色 5 维是否全部填充？是否有主角-反派对比？
- [ ] ★ 角色语言指纹库：表格列名是否原样使用了"句长签名/句式签名/词汇签名/转折签名/功能签名"5 个名称？
- [ ] ★ 句式模式库：是否至少提取 5 种句式？每种是否含四元组 + 原文案例 + 替换规则？
- [ ] ★ 句式模式库：分类名称是否使用了规定的 8 种标签（压制式/逆转式/揭露式/递进式/留白式/引入式/收束式/复调式），而非自行命名？
- [ ] ★ 句式模式库：每种句式的"替换规则"是否明确区分了"可替换"和"不可改"+ 给出跨赛道映射示例？
- [ ] ★ 阶段三·互动模式分析：是否按五类统计了频次/占比/高频载体/阶段变化？
- [ ] ★ 阶段三·互动模式分析：是否输出了结构失衡诊断数据（非写作模板）？
- [ ] ★ 阶段三·互动模式分析：高频载体是否显式标记为"供下游禁用"？

### 阶段四产出检查（模板合成 + 最终质量自检）
- [ ] 剧情模板是否可实际用于生成新章纲（不只是泛泛描述）？
- [ ] ★ 参数化标记是否严格三层统一？使用是否一致？
- [ ] ★ 全局变量清单表是否已写入附录？
- [ ] ★ 原著辨识度清单（§X）是否至少标注 5 个标志性事件节点（极高≥2）+ 2 个高频模板？是否有 X.3 不可复用度评分表？
- [ ] 全局：每个模块是否至少有 3 个原文章节案例佐证？
- [ ] 全局：全书结构地图、抽样章拆解记录是否完整写入附录？

## 二、跨风格提取框架

### 2.1 社会语言层次模型（§14）

划分至少 3 层（顶层/中层/底层），每层提取 4 个跨风格维度：

| 维度 | 提取内容 | 跨风格含义 |
|------|---------|-----------|
| **自称模式** | 该层角色如何称呼自己？(命令式/谦卑式/平等式) | 权力信号的语法载体 |
| **句法特征** | 句长倾向？命令句/疑问句/陈述句的比例？ | 权力在语言中的节奏体现 |
| **对话控制权** | 谁决定对话开始/结束？谁能打断谁？ | 社会地位在对话流程中的映射 |
| **情感泄漏** | 情绪失控时语言发生什么变化？(句长骤变/称呼切换/沉默) | 权力面具的裂缝模式 |

输出格式（必须使用此表格，每层一行）：
```
| 层级 | 自称模式 | 句法特征 | 对话控制权 | 情感泄漏 |
|------|---------|---------|-----------|---------|
| 顶层 | (提取)   | (提取)   | (提取)     | (提取)   |
| 中层 | (提取)   | (提取)   | (提取)     | (提取)   |
| 底层 | (提取)   | (提取)   | (提取)     | (提取)   |
```

最低要求：至少 3 层、每层 4 维全填充、每维度至少 1 个原文案例。

### 2.2 角色语言指纹库（§15）

提取对象：主角 + 至少 1 个主要反派 + 至少 1 个主要盟友（最低 3 人，最多 8 人）

每个角色 5 个固定维度：

| 维度 | 名称 | 提取内容 | 跨风格通用性说明 |
|------|------|---------|----------------|
| 1 | **句长签名** | 该角色对话的平均句长 ± 标准差 | 所有语言都有句长概念 |
| 2 | **句式签名** | 最高频的 3 种句式类型（祈使/疑问/陈述/感叹/省略/反问/条件） | 句式类型是通用语法范畴 |
| 3 | **词汇签名** | 最高频的 3-5 个专属词/口头禅/称呼方式 | 所有角色都有标志性用词 |
| 4 | **转折签名** | 情绪剧变时句法如何改变（句长变化方向、称谓是否切换、是否沉默） | 情绪对语言的扰动是跨文化现象 |
| 5 | **功能签名** | 该角色对话的主要功能偏向（推进剧情/展示性格/传递信息/制造氛围）及大致比例 | 对话功能分类不依赖内容 |

输出格式（必须使用此表格，每个角色一行）：
```
| 角色 | 句长签名 | 句式签名(Top3) | 词汇签名(Top3-5) | 转折签名 | 功能签名 |
```

必须产出至少 1 组主角-反派的指纹对比分析。

### 2.3 句式模式库（§16）

必须覆盖的 8 种句式分类（固定标签，禁止自行重命名）：

| 序号 | 句式名称 | 通用识别特征 | 跨风格适用场景 |
|------|---------|------------|-------------|
| 1 | **压制式** | 角色A用身份/信息/武力对B建立优势的句式 | 打脸/装逼/收服/初次亮相 |
| 2 | **逆转式** | 从劣势到优势的瞬间翻转句式 | 爽点高潮/绝境翻盘 |
| 3 | **揭露式** | 信息从隐藏到公开的句式 | 布局收束/身份揭晓/真相大白 |
| 4 | **递进式** | 情绪/信息/冲突逐步加码的句式 | 铺垫积累/紧张升级 |
| 5 | **留白式** | 用反应/沉默/第三人称叙事替代直接表达的句式 | 情感高潮/政治博弈/诀别场景 |
| 6 | **引入式** | 新角色/新场景/新信息的首次亮相句式 | 场景切换/角色登场/信息释放 |
| 7 | **收束式** | 一个事件/情绪/场景的收尾句式 | 章末收尾/转场过渡/余韵 |
| 8 | **复调式** | 同一时刻多线并进或多人反应交叉的句式 | 大高潮/群像场景/势力博弈 |

每种句式的四元组输出格式（强制执行）：
```
句式 #N：[名称]
  结构：
      节奏：    (短/中/长句的交替节奏模式)
      视角锚定：(谁在看/在想/在感受)
      信息释放：(一次性/分步/延迟)
      情感走向：(上扬/下沉/先抑后扬/先扬后抑)
  使用场景：(该句式常出现在什么剧情节点)
  叙事效果：(对读者产生什么心理影响)
  替换规则：
      可替换：(列出 2-3 项)
      不可改：(列出 2-3 项)
      跨赛道映射示例：(1 个具体示例)
  原文案例：(1 个典型段落，标注章节)
```

最低要求：至少提取 5 种句式、每种含四元组+案例+替换规则。

## 三、参数化标记系统规范（§19）

白皮书全文中必须统一使用以下三层标记，不得混用：

| 层级 | 标记语法 | 定义 | 数量约束 | 示例 |
|------|---------|------|---------|------|
| 第一层 | `{底层逻辑}` | 不可修改的核心叙事规则 | 全文 5-8 条 | `{预知未来}` `{身世隐藏}` `{等级不可跨越}` |
| 第二层 | `[表层变量]` | 本世界内的具体设定，可替换但不改变底层逻辑 | 按需标记 | `[清朝]→[架空王朝]` `[曹家]→[林家]` |
| 第三层 | `{{迁移变量}}` | 跨赛道迁移时一键替换的变量 | 仅出现于可衍生性评估章节 | `{{主角能力载体}}` `{{世界观时代}}` |

使用规则：
- `{底层逻辑}` 必须在第一次出现时附一句话解释——为什么这是不可修改的
- `[表层变量]` 在人物/地点/组织/关键物品首次出现时标记
- `{{迁移变量}}` 仅用于"可衍生性评估"章节，标注迁移时的替换建议

### X.1 标志性事件（不可原样复用的高辨识度节点）

|----------|:---------:|---------|-------------|
| （事件A） | 极高 | （为何是签字记忆） | （可选的替代方式） |
| （事件B） | 极高 | | |
| （事件C） | 高 | | |
| ... | | | |

至少标注 5 个节点（极高 ≥2 个、高 ≥3 个）

### X.2 高频套路模板（易被无意识复用的叙事模式）

至少列出 2 个高频套路模板，每个含使用频率和仿写规避建议。

### X.3 不可复用度评分

- 极高辨识节点 ×1 → 签约审核"换皮抄袭"一票否决 命中概率：90%
- 高辨识节点 ×3（在连续 10 章内）→ 命中概率：50%
- 高辨识节点 ×1（有实质性变形）→ 安全，不触发
