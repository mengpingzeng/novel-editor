## 变更一：D1 字数合规评分调整

### 变更文件

`project-agents-template/.opencode/agents/quality_reviewer.md`

### 变更概述

收紧字数满分区间，降低绝对上限，使字数控制更为严格。

### 变更对比

| 项目 | 原值 | 新值 |
|------|------|------|
| 满分区间 | [1500, 2500] | [1500, 2000] |
| 绝对下限 | 1000 字（不变） | 1000 字 |
| 绝对上限 | 3000 字 | 2500 字 |
| 下过渡区间公式 | `5 × (字数 - 1000) / 500` | 不变 |
| 上过渡区间公式 | `5 × (3000 - 字数) / 500` | `5 × (2500 - 字数) / 500` |
| 硬性不合格 | < 1000 或 > 3000 | < 1000 或 > 2500 |

### D1 得分规则（新）

| 实际字数 | D1 得分 | 说明 |
|----------|---------|------|
| < 1000 | 整篇 0 分 | 硬性不合格，所有维度归零 |
| [1000, 1500) | `5 × (字数 - 1000) / 500` | 线性过渡 |
| [1500, 2000] | **5（满分）** | 字数合规满分 |
| (2000, 2500] | `5 × (2500 - 字数) / 500` | 线性过渡 |
| > 2500 | 整篇 0 分 | 硬性不合格，所有维度归零 |

### 计算示例（新）

| 字数 | D1 得分 | 公式 |
|------|---------|------|
| 1200 | 2.0 | `5 × (1200-1000)/500` |
| 1400 | 4.0 | `5 × (1400-1000)/500` |
| 2000 | 5.0 | 满分区间 |
| 2400 | 1.0 | `5 × (2500-2400)/500` |
