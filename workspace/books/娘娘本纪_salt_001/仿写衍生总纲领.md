## 一、核心定位

### 1.1 项目概要
将女娲（神性创世母神·洪荒流）的主视角、力量体系、关系网络、剧本范式完整映射为**现代都市隐秘江湖框架**。主角变体为赘婿神医（林阳/叶青云），力量体系从古典修仙替换为**武道医道双修**（明劲→道主），保留全部爽点公式、进阶公式、节奏模型与四段式章节结构，适配番茄小说高爽点密度、强对话驱动的都市赘婿神医赛道。

### 1.2 赛道定位
- **子分类**: 赘婿神医流（隐藏大佬逆袭）
- **预计总章节**: 120章
- **预计总字数**: 180万-240万字
- **核心读者**: 男性向，25-45岁，喜欢逆袭打脸、扮猪吃虎、医道传承题材

## 四、爽点体系

### 4.1 核心公式
总爽感 = Σ(力量成长×0.3 + 地位跃升×0.25 + 情感满足×0.2 + 智慧碾压×0.15 + 宝物获得×0.1) × 节奏系数

### 4.2 爽点分类与频率
| 类型 | 频率 | 都市版案例 |
|------|------|-----------|
| 境界突破 | 每8-12章1次 | 第3章明劲→暗劲，第8章暗劲→化劲 |
| 宝物获得 | 每5-8章1次 | 第2章神农医典残卷，第6章阴阳五行针 |
| 打脸反杀 | 每10-15章1次 | 第6章当众治好绝症，第9章拿出千年野山参 |
| 收徒立派 | 每15-20章1次 | 第14章收徒传医，第21章收小七 |
| 悟道讲道 | 每10-15章1次 | 第3章引动天地元气异象，第15章医学研讨会 |
| 情感羁绊 | 每5-8章1次 | 第1-2章妻子态度转变，第15章为救妻子自损修为 |

### 4.3 密度标准
- 总爽点事件预估：360-480次（120章）
- 平均每章爽点：3-4次
- 小爽点：每1-2章
- 中爽点：每3-5章
- 大爽点：每8-10章

## 六、文风标准

### 6.1 数据指标
- 平均句长：15-25字/句
- 对话占比：40%-50%
- 叙事视角：第三人称有限视角（锁定主角）
- 情感渲染：动作+对话+简短心理（快节奏交替）
- 古典词汇密度：5%-10%
- 平台适配：每500字内必有钩子/反转/信息差

### 6.2 句式模板
- **环境描写**: "放眼望去" + [都市元素] + [动态动词] + [修饰前缀] + {现代感四字词组}
- **战斗描写**: [动作主体] + [爆发性动词] + [招式/内劲名] + [效果描述] + {胜负结果}
- **情感表达**: [情绪触发事件] + [身体反应] + [简短内心独白/誓言] + {收尾}
- **悟道讲道**: [引子:古医有云] + [医道/武道命题] + [辩证阐述] + [顿悟结论]

### 6.3 文体配比
- 叙述（60%）：现代叙事50% + 简洁描写10%
- 对话（30%）：口语化25% + 文雅5%
- 描写（10%）：现代白描8% + 少量文雅2%

保留"陡然""蓦然""须臾"等少量高频节奏词，删除"只见""却道"等古风转场词，替换为"突然""就在这时""下一秒"等现代节奏词。

## 七、平台适配

### 7.1 番茄小说适配要求
- 前三章：身份展示+金手指出现+第一次小高潮打脸
- 章节长度：3000-5000字
- 对话比例：≥40%
- 爽点密度：每章≥3小爽点 + 每3章1中爽点 + 每10章1大爽点
- 信息差运用："所有人都以为他是废物"信息差打脸
- 更新频率：日更2章，章末必留悬念钩子

### 7.2 卷计划
- 第一卷（第1-30章）：赘婿觉醒
- 第二卷（第31-60章）：势力崛起
- 第三卷（第61-90章）：古族博弈
- 第四卷（第91-120章）：终极对决

## 八、禁止改动底层逻辑

1. 宇宙层级结构的嵌套模型（高层→中层→底层三层连通架构）

3. 章节节奏四段式模型（开端→铺垫→高潮→余韵 百分比分布）
4. 进阶公式（天赋×功法×(1+资源)×悟性−瓶颈）
5. 力量层次换算的指数级比例关系
6. 角色模板的五维性格参数雷达结构
7. 关系网的辐射中心式拓扑结构
8. 开局/发展/高潮/收尾四大范式步骤
