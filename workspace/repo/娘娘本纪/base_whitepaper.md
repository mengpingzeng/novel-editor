**原文案例**：第3章——"人法地，地法天，天法道，道法自然"；第33章讲道——"有物混成，先天地生，寂兮寥兮，独立不改"。

### 5.3 古文与现代文配比

```
叙述界面(60%)   =  古风词汇(30%) + 现代叙事(30%)

```

**案例佐证**：全篇大量使用"只见""蓦然""陡然""须臾"等古风临场词，但对话偏口语化如"为什么""好漂亮""hold不住"等现代词汇混用。
