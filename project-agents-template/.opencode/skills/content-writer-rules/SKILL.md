## 一、章内节奏结构（强制遵循）

严格按照总纲领"节奏模型"章节定义的五段式结构分配篇幅，参考以下字数分布：

```
假设目标字数 = T（从总纲领"平台适配"读取，如 2000 字）：
┌──────────┬────────────┬──────────────────────────────────────┐
│ 段落      │ 占比        │ 字数参考（T=2000时）                │
├──────────┼────────────┼──────────────────────────────────────┤
│ ① 开场   │ 0~10%      │ 约 150~200 字                        │
│ ② 铺垫   │ 10~50%     │ 约 800~1000 字                       │
│ ③ 转折   │ 50~75%     │ 约 400~500 字                        │
│ ④ 高潮   │ 75~92%     │ 约 300~400 字                        │
│ ⑤ 收尾   │ 92~100%    │ 约 150~200 字                        │
└──────────┴────────────┴──────────────────────────────────────┘
```

**节奏红线**：
- ❌ 铺垫占比超过 50% → 判定为节奏拖沓
- ❌ 开场超过 3 段还没进入正题 → 判定为慢热
- ❌ 收尾没有钩子 → 判定为结构不合格
- ❌ 收尾段占比超过 15%（硬上限）→ 判定为收尾冗长
- ❌ 完整的前世画面/大段心理独白/新信息释放放入收尾段
- ❌ 出现超过 150 字的单段连续文本
- ❌ 连续 3 段无句号断点

### 段落密度与短段落要求（防拥挤排版）

```
✅ 每段 1~3 句，句号(。)断句
✅ 段落间空行分隔
✅ 每千字段落数 ≥ 8 段
✅ 对话段落 1~2 句为主，叙述段落 ≤3 句
❌ 单段超过 150 字
❌ 连续段落不用句号而用逗号串联
❌ 大段说明文（>200字纯叙述无对话无动作）
```

## 二、文风量化指标自检

| 量化指标 | 目标范围 | 自检方式 |
|---------|---------|---------|
| 平均句长 | 15~25 字 | 统计全文总字数 ÷ 句数 |
| 对话占比 | 55%~65% | 引号内字数 ÷ 总字数 |
| 段落长度 | 每段 2~4 句 | 超过 5 句的段落不超过 2 处 |
| 短段落占比 | ≥ 50%（≤3句的段落 / 总段落数） | 计数 |
| 句号密度 | ≥ 2.5（句号数 × 1000 ÷ 总字数） | 移动端网文标准 |
| 段落密度 | ≥ 7段/千字 | 低于此值即判定排版过挤 |
| 叙事视角 | 严格第三人称有限 | 始终跟随主角视角 |

**文风红线**：
- ❌ 出现连续超过 5 句的纯叙述段落（无对话无动作）
- ❌ 出现超过 200 字的背景/行业说明段落
- ❌ 出现主角视角之外的场景切换

### 角色语言指纹落地

从总纲领"文风句式"章节读取"角色语言指纹库"，写作时必须遵守：
1. 指纹一致性：每个出场角色的对白必须符合其 5 维指纹
2. 主角-反派差异：确保主角与反派的转折签名形成鲜明对比
3. 身份对话梯度：不同社会层级的角色对话必须体现梯度差异
4. 写完后自问：遮住角色名，能否仅从对白判断出是谁在说话？

### 句式模式落地

从总纲领"文风句式"章节读取"句式模式库"，必须引入变异率，禁止 100% 遵循原著句式模式：
- **70%** 概率：优先使用首选句式，遵守"不可改"规则
- **20%** 概率：微调——保持句式骨架但调整 1 个参数
- **10%** 概率：替换——使用次选句式或平台原生句式

连续章节句式轮换：同一句式不得在连续 3 章中作为首选句式出现。

## 三、章首衔接三要素（强制）

| 衔接方式 | 写法示例 | 适用场景 |
|---------|---------|---------|
| ① 上章回音 | "上一章的事件过去后，X 的心情久久不能平静……" | 连续剧情 |
| ② 新冲突直入 | "第二天一早，苏逸被怒吼声吵醒——'给我滚出来！'" | 切换时间线/新冲突 |
| ③ 悬念延续 | "那个神秘电话里的人，竟然是他？" | 上章悬念延续 |

**禁止**：用大段环境描写开场（"清晨的阳光透过窗帘洒在地板上……"）

## 四、钩子设计模板（收尾强制使用）

章末收尾段必须选用以下 3 种钩子模板之一：

| 钩子类型 | 模板 | 示例 | 效果 |
|---------|------|------|------|
| ① 悬念钩 | 关键信息中断 + 下章预告 | "门开了。进来的那个人，让他瞳孔骤缩——" | 制造好奇 |
| ② 金句钩 | 主角霸气台词 + 停顿留白 | "苏逸端起茶杯，笑了笑：'在座的各位，有一个算一个——'" | 适合截图传播 |
| ③ 反转钩 | 局势突然逆转 + 后果暗示 | "就在所有人以为尘埃落定时，手机震了一下——" | 打破预期 |

钩子设计原则：
- 必须在最后 1~3 段内完成
- 禁止以"欲知后事如何，请听下回分解"这种过时写法
- 钩子类型轮换约束：连续 3 章不得使用同一类型钩子
- 钩子载体多样化：若前 5 章已有 ≥2 次"设备推送信息"类载体，须更换为物理线索触发

## 五、金句设计指导

| 规则 | 要求 |
|------|------|
| 频率 | 每 5~8 章至少设计 1 句可传播的高能台词 |
| 长度 | 不超过 30 字，简短有力 |
| 位置 | 高潮段落末尾或打脸场景最高点 |
| 风格 | 符合角色人设，适合社交媒体截图传播 |
| 禁用 | 尴尬的强行装逼、不符合角色性格的台词 |

## 六、情感缓冲机制

| 情感线类型 | 频率要求 | 作用 |
|-----------|---------|------|
| 爱情线 | 每 3~5 章至少 1 次 | 展现柔软一面 |
| 亲情线 | 每 5~8 章至少 1 次 | 长辈/暗中关怀 |
| 友情线 | 每 5~8 章至少 1 次 | 信任建立 |

情感锚点角色出场保障：
- 从总纲领/人设表中识别"情感锚点角色"
- 该角色每 3 章至少出场 1 次（≥1 段互动）
- 互动内容须能体现主角的非战斗面（温情/信任/柔软/日常）

## 七、标签场景落地清单

| 标签 | 本章必须完成的写作动作 |
|------|----------------------|
| 扮猪吃虎 | □ 主角表面弱势形象维持 □ 基于信息差的打脸 □ 对手轻视→震惊反应 |
| 赘婿逆袭 | □ 家庭地位压制场景 □ 触发逆袭动机的冲突 |
| 商战重生 | □ 前世记忆/商业预判运用 □ 商业场景决策或交锋 |
| 隐藏大佬 | □ 暗中操作或布局 □ 真实实力继续隐藏 |
| 信息差碾压 | □ 核心打脸手段展示 □ 四步闭环：自信→点破→验证→震惊 |

## 八、写作顺序建议

```
第〇步：读摘要   → 读取伏笔状态滚动摘要
第一步：定钩子     → 想好本章结尾用什么钩子
第二步：定高潮     → 本章最燃的打脸/反转场景
第三步：定铺垫     → 为高潮做信息铺垫
第四步：定开场     → 用什么方式切入
第五步：填充细节   → 对话、心理、动作、配角反应
```

## 九、AI 重复标记消除（强制自检）

### 9.0 章纲引用清洗铁则（零容忍）

**绝对禁止在正文叙事中出现"第X章"（X为数字）的章节编号引用。**

转换规则表：

| 章纲中的引用 | ❌ 禁止写法 | ✅ 正确写法 |
|-------------|-----------|-----------|
| "第3章问道碑拆解道则之后" | "第3章问道碑拆解道则之后" | "古剑冢那一战之后" |
| "从第1章站在山门外受检开始" | "从第1章站在山门外受检开始" | "自青云剑宗山门被弃那天起" |
| "第4章他放走的密报者" | "第4章他放走的密报者王庆" | "那个被他放走的密报者王庆" |

### 9.0-B 标签元数据禁令（零容忍）

**绝对禁止在正文任何位置出现标签/分类元数据。** 违禁内容："标签：" 后跟任何文字、"标签覆盖"、"本章标签"、"分类标识"。

### 9.1 句式重复禁令

以下句式模式在同一章内不得超过 2 次，连续 3 章内不得超过 3 次：

| 禁令类型 | 典型违例 | 替换策略 |
|---------|---------|---------|
| 主角名+否定动词固定句式 | "凤华没有回应""凤华没有动" | 替换为差异化动作/对话回应/环境反应 |
| 同一生理反应描写 | "瞳孔微缩""瞳孔骤缩" | 轮换池：眉头微蹙/喉结滚动/呼吸一滞/指尖微颤/目光凝住 |
| 同一情绪形容词 | "冷冽""凛冽""寒意" | 同一情绪词每章最多用 1 次 |
| 比喻套路化 | "目光如寒潭""声音平静得像一潭死水" | 比喻须每章变换喻体来源 |
| **"不是X——是Y"句式（绝对禁止·零容忍）** | "不是变色——是碎""不是快了——是颠覆了" | 直接陈述/用动作展示/用对话替代/因果倒装/否定疑问句 |
| 反应场景模板化 | "全场鸦雀无声""全场死一般寂静" | 替换为具体个体反应 |
| 原著高频词未变异 | 直接移植"瞳孔一缩""倒吸凉气""场中寂静" | 须执行白皮书§6.11 变换策略 |
| 同一描写词全本高频复用 | 同一描写词在 20 章内 >40 次 | 建立 5+ 个同义词轮换池 |
| 架空背景现代术语泄露 | 仙侠中出现"HRC五十八""DNA""蛋白质" | 替换为感官化/经验性描述 |

### 9.1-B 比喻密度硬上限

- 每章明喻（"像……一样/般"）≤ 3 个
- 连续 3 章不得使用同一类喻体来源（自然意象/器物意象/动物意象/抽象概念）

### 9.2 描写轮换池机制

- **冷硬形象表达池**（≥6 个，每章选 ≤2 个）：面无表情/眼神淡漠/嘴角平直/神色不动/眉梢未挑/气息内敛
- **震惊反应表达池**（≥6 个，每章选 ≤2 个）：瞳孔微缩/呼吸一滞/指尖微颤/喉结滚动/后背绷直/目光凝住
- **杀意/压迫表达池**（≥6 个，每章选 ≤2 个）：周身气压骤降/目光如淬了毒/声音冷了三分/杀气凝成实质/空气仿佛冻结/令周围人本能后退

### 9.3 写后强制自检（提交前必勾）

```
□ 全文搜索"没有/不+动词"句式，计数 ≤2 次/章
□ 全文搜索"瞳孔+缩"描写，计数 ≤1 次/章
□ 全文搜索同一情绪形容词，同一词计数 ≤1 次/章
□ 检查比喻句喻体，同章内无重复喻体
□ 明喻总数 ≤3 个/章，且无连续出现
□ 冷硬形象表达池取用 ≤2 个
□ 震惊反应表达池取用 ≤2 个
□ "不是X——是Y"句式：计数须为 0次/章（零容忍）
□ 章纲引用残留：正文叙事中不得出现"第X章"，计数须为 0（零容忍）
□ 全文搜索"全场|满堂|满殿"+(寂静|鸦雀|安静|死寂)，命中即替换
□ 全文搜索"瞳孔一缩|倒吸凉气|场中寂静"，若命中须做变异处理
□ 全文搜索同一描写词，若 20 章累计 >40 次则建立轮换池
□ 生产元数据清除：全文搜索">>> 模型自检""【自检】""字数预估"，发现即删除
□ 标签元数据清除：全文搜索"标签[:：]"，计数须为 0
□ 段落密度：单段 >150字不超过 0 处，段落数/千字 ≥ 7
□ 句号密度：句号数量 × 1000 ÷ 总字数 ≥ 2.5
□ 替换词轮换检查：本章替换词在已写章节中是否已出现≥3次
□ 金手指启动方式检查：本章触发方式是否与上一章相同
□ 现代词汇转化检查：架空背景中是否出现现代科技/工业/学术词汇
□ 环境侧写元素轮换：本章环境侧写是否使用了与前一章相同的元素组合
```

### 9.4 写完即清除·文件收尾铁律

每章正文 Write 完毕前，必须执行文件末尾检查：
1. 最后一行必须是小说正文——不是元数据
2. 禁止在文件末尾附加任何"标签："行、"本章覆盖"行、"字数统计"行
3. 读回刚写入的文件最后 5 行，确认无元数据泄漏

### 9.5 替换词轮换机制（防"每修一tic引入一新tic"）

轮换铁则：
1. 从盐值 fingerprint_variation_table 选取替换词时，同一替换词在 20 章内使用不得超过 3 次
2. 若 variants 数组有 ≥3 个替换词 → 每 3 章轮换一次
3. 若自行创造替换词 → 在纪要中记录，累计 ≥3 次后强制换词
4. 环境侧写须从盐值 environmental_side_writing_pool 中轮换选取，同一元素组合不得连续 2 次使用

### 9.6 金手指启动方式变体库

铁则：
1. 从盐值 golden_finger_spec.presentation_variants 中选取，同一 variant 在 10 章内使用不得超过 3 次
2. 若 presentation_variants 数量 <5 → 须自行扩展变体
3. 禁止连续 2 章使用完全相同的"触发→展开→获得信息"三段式结构
4. 同一触发方式在同一章内使用不得超过 1 次

### 9.7 现代词汇转化指引（架空背景专用）

| 现代词汇 | ❌ 禁止 | ✅ 转化方向 |
|---------|---------|-----------|
| 数据/数据库 | "这些数据从哪儿来的" | "这些详情/依据/性状" |
| 交互网络 | "药性交互网络" | "药性交互之理/相生相克之网" |
| 效率 | "融合效率" | "融合之效/融合速度" |
| 系统/程序 | "系统运行/程序启动" | "阵法运转/法诀催动" |
| 参数/变量 | "调整参数" | "调整火候/调控药引" |
| 检测/扫描 | "检测到异常" | "察觉到异样/感知到异常" |

## 十、配角差异化约束（防工具人化/脸谱化）

### 10.1 配角辨识标记（强制）

每个出场 ≥1 场的命名配角（含小反派），必须在首次出场时赋予至少 1 个 distinctive 辨识标记：

| 辨识标记类型 | 示例 | 约束 |
|-------------|------|------|
| 生理特征 | 左手缺一指/总是舔嘴唇/右眼有旧疤 | 须贯穿所有出场 |
| 行为习惯 | 说话前先咳嗽一声/紧张时拨弄腰带 | 须贯穿所有出场 |
| 语言特征 | 用"某家"自称/句末带"罢了" | 须与角色语言指纹一致 |
| 物品关联 | 总随身带一杆烟枪/腰间挂一把旧钥匙 | 须自然展示 |

### 10.2 配角独立动机落地

- 若章纲为配角标注了 `independent_motivation`，正文须至少有 1 处展示其追求自身目标的行为
- 若章纲未标注但该配角出场 ≥3 场，须为其设计 1 句话暗示其个人追求/困境/执念

### 10.3 小反派去脸谱化

- 小反派须至少展示 1 个非面子之争层面的动机或困境
- 同一批次（5 章）内出现的 ≥2 个小反派须有差异化性格特征

## 附：重写模式

当 mode=rewrite 时：
1. 以现有初稿为基础修改，而非从零重写（除非总分 <40）
2. 逐条对照纪要中的"逐条修改建议"修正
3. 优先解决 D2（文风仿写）和 D3（爽点落地）维度的扣分项
4. 保持章纲核心事件不变，不得自行扩展支线
5. 重写后重新自检
6. 总分 <40 时：判定为结构性失败，从章纲重新撰写全章
